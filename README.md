# cpp-primer-5th
cpp primer 5th answers

## requirements

- CMake 3.1+
- VS 2015+

## build on windows

- clone this repo
- open local repo path
- use CMake GUI build the vs project. Or open prompt 
  in current repo directory(`shift + right-click`), and:
  ```shell
  %repo-path%> mkdir build
  %repo-path%> cd build
  %repo-path%> cmake ..
  ```
- open with vs2015+
- enjoy
